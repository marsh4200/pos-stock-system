# Changelog

## v3.5.3 — 2026-05-28

### Fixed
- **Product search now finds items by barcode.** On the Products screen, searching (or scanning into the search box) only matched name, SKU, and category — a barcode would never return its product even though barcodes were stored against it. The filter now also checks every barcode linked to each product, so a scanned or typed barcode finds the item.
- Search box placeholder updated to "Search by name, SKU, category, or barcode…" to reflect this.
